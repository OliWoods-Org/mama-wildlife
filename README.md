> **1 million species face extinction.** Wildlife crime is a **$23B/year industry.** Most people want to help but don't know how. This tool makes conservation accessible to everyone.

## The Solution

5 agents connecting people to conservation action.

## Agents

**SpeciesTracker** — Endangered species database, population trends, habitat maps (IUCN Red List)
**ReportingAgent** — Report wildlife crime, poaching, habitat destruction to authorities
**VolunteerMatcher** — Conservation volunteer opportunities by location
**EcoTourismGuide** — Responsible wildlife tourism vs exploitative attractions
**EducationAgent** — Species facts, conservation status, actionable steps

## Safety

Report wildlife crime to US Fish & Wildlife Service: 1-844-FWS-TIPS

## Data Sources

IUCN Red List API, eBird API, iNaturalist API, FWS enforcement data
